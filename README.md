# HTML, CSS & JS Lab 🚀

HTML, CSS & JS Lab is a modern, interactive web-based learning platform for frontend development. It provides a hands-on environment where users can learn HTML, CSS, and JavaScript through a series of progressive challenges with real-time feedback and automated testing.

## ✨ Features

- **🎓 Interactive Curriculum**: 40+ carefully crafted challenges covering everything from basic HTML tags to advanced JavaScript concepts and mini-projects.
- **⚡ Real-time Preview**: See your changes instantly in a live preview window as you type.
- **🧪 Automated Testing**: Built-in validation engine that checks your code against specific requirements for each challenge.
- **💻 Professional Editor**: Integrated Monaco Editor (the engine behind VS Code) for a premium coding experience with syntax highlighting and autocompletion.
- **💾 Progress Tracking**: Automatically saves your code and progress to local storage, so you can pick up right where you left off.
- **🎨 Code Formatting**: One-click code beautification using Prettier.
- **🔗 Shareable Solutions**: Generate unique URLs to share your solutions with others.
- **🌙 Modern UI**: A sleek, dark-themed interface designed for focus and productivity.

## 🛠️ Tech Stack

- **Core**: [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Editor**: [Monaco Editor](https://microsoft.github.io/monaco-editor/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Formatting**: [Prettier](https://prettier.io/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/pratham-prog861/html-css-js-lab.git
   cd html-css-js-lab
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📖 How to Use

1. **Select a Challenge**: Choose a topic from the curriculum sidebar on the left.
2. **Read Instructions**: Review the challenge description and requirements in the middle panel.
3. **Write Code**: Use the HTML, CSS, and JS tabs to implement your solution.
4. **Run Tests**: Click the "Run Tests" button to verify your solution.
5. **Format & Share**: Use the toolbar to format your code or generate a shareable link.
